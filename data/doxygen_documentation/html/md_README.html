Description générale

Ce projet est réalisé dans le cadre de l'UE de Projet de Programmation en M1 Informatique à l'université de Bordeaux. Le responsable est Monsieur Philippe Narbel, et le projet est suivi par Madame Wahiba Larbi. Le projet ci-présent se présente comme un générateur de grilles de mots croisés avec interface utilisateur.

Auteurs

Les auteurs de ce projet sont Clément Albanel, Hugo Devidas, Emma Auzi, Nathanael Alves, Yohan Bornes et Sylvain Coudougnan, tous issus du parcours Génie logiciel.

Installation

Le projet nécéssite l'utilisation de l'API Qt pour la gestion de l'interface graphique. Celle-ci possède une version open-souce, que nous avons utilisée.

1) Installation de Qt (Linux version - testé sur Ubuntu 20.04): sudo apt-get install qtbase5-dev && sudo apt-get install qtdeclarative5-dev && sudo apt install qttools5-dev

2) Lancement de l'application a) cd src/build b) cmake .. c) make d) ./CrossedWord term ../../data/word_database.txt ../../data/pattern/easy/custom1.txt (mode terminal) e) ./CrossedWord (mode graphique)
